# OpenRouter Setup for Vision Features

## Get Free OpenRouter API Key

1. Visit: https://openrouter.ai/
2. Click "Sign In" (top right)
3. Sign in with Google/GitHub (free, no credit card)
4. Go to: https://openrouter.ai/keys
5. Click "Create Key"
6. Copy the key (starts with `sk-or-v1-`)

## Add to .env File

```bash
# Add this line to your .env file
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

## Free Credits

- OpenRouter gives **$1 free credit** on signup

## Alternative: Use Without Vision

If you don't want to set up OpenRouter, the app will still work but image analysis will provide a template instead of AI analysis.
